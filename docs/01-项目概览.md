# 01 - 项目概览

## 项目简介

本仓库通过从 npm 公开包 `@anthropic-ai/claude-code`（v2.1.88）中提取 source map（`cli.js.map`），还原了 Claude Code CLI 的完整 TypeScript 源码。

**Claude Code** 是 Anthropic 出品的终端 AI 编程助手，能够：
- 理解整个代码仓库
- 编辑文件、运行终端命令
- 处理 Git 工作流（提交、PR、分支管理）
- 执行多步骤编程任务
- 通过多 Agent 协作处理复杂项目

## 还原方法

### 提取脚本：`extract-sources.js`

```javascript
// 核心逻辑：
// 1. 读取 package/cli.js.map（source map 文件）
// 2. 遍历 rawMap.sources 和 rawMap.sourcesContent
// 3. 将每个源文件写入 restored-src/ 目录
const rawMap = JSON.parse(fs.readFileSync('package/cli.js.map', 'utf8'));
for (let i = 0; i < rawMap.sources.length; i++) {
  const content = rawMap.sourcesContent[i];
  if (!content) continue;
  fs.writeFileSync(fullPath, content, 'utf8');
}
```

## 技术栈

| 技术 | 用途 |
|------|------|
| **TypeScript** | 主语言，全部源码均为 `.ts` / `.tsx` |
| **React** | UI 框架（配合 React Compiler） |
| **Ink** | 终端 UI 渲染引擎（React → Terminal） |
| **Zod** | 运行时 Schema 验证（工具输入/输出） |
| **Commander.js** | CLI 参数解析 |
| **Anthropic SDK** | API 客户端（支持直连、Bedrock、Vertex AI） |
| **ES Modules** | 原生 ESM 模块系统 |
| **GrowthBook** | 特性开关（Feature Flags） |

## 运行环境

- **Node.js**: 18.0.0+
- **平台**: macOS、Linux、Windows
- **内嵌工具**: ripgrep（代码搜索）、audio-capture（语音输入）

## Claude Code 的核心能力

### 1. 代码理解与编辑
- 读取、搜索、编辑项目中的文件
- 支持 glob 模式和正则搜索
- 智能代码差异（diff）展示

### 2. 终端命令执行
- 在安全沙箱中运行 Bash/PowerShell 命令
- 命令安全分类（只读/破坏性检测）
- 后台任务支持

### 3. Git 工作流
- 提交、分支、PR 管理
- 代码审查
- 变更追踪

### 4. 多 Agent 协作
- 子 Agent 派生（AgentTool）
- 团队模式（Team）
- Worktree 隔离（Git Worktree 独立工作区）

### 5. 外部集成
- MCP（Model Context Protocol）工具集成
- LSP（Language Server Protocol）代码智能
- 网页抓取与搜索
- 插件系统与技能系统

### 6. 用户交互
- 终端 UI（Ink 渲染）
- 语音输入/输出
- Vim 编辑模式
- AI 伴侣精灵（Buddy）
