# 17 - 特色子系统

## 概述

Claude Code 包含多个独特的子系统，为终端体验增添特色功能。本文档介绍伙伴系统、Vim 模式、语音输入三个最具特色的子系统。

## Buddy 伙伴系统

### 什么是 Buddy？

一个确定性生成的虚拟伙伴精灵，每个用户根据其 ID 获得唯一的伙伴。

- **门控**：`feature('BUDDY')` 编译时开关
- **目录**：`buddy/`（6 个文件）

## Vim 模式

### 什么是 Vim 模式？

纯函数式 Vim 键绑定模拟器，让用户在 Claude Code 的输入框中使用 Vim 快捷键编辑文本。

- **切换**：`/vim` 命令
- **目录**：`vim/`（5 个文件）
- **特点**：纯函数、无副作用、状态机架构

### 状态机

```typescript
// vim/types.ts
type VimState =
  | { mode: 'INSERT'; insertedText: string }
  | { mode: 'NORMAL'; command: CommandState }

type CommandState =
  | { type: 'idle' }                // 空闲
  | { type: 'count'; digits }       // 输入数字
  | { type: 'operator'; op; count } // 等待 motion
  | { type: 'operatorCount'; ... }  // 操作符后输入数字
  | { type: 'operatorFind'; ... }   // 操作符 + f/t
  | { type: 'operatorTextObj'; ... }// 操作符 + 文本对象
  | { type: 'find'; ... }          // f/F/t/T
  | { type: 'g'; ... }             // g 前缀
  | { type: 'replace'; ... }       // r 替换
  | { type: 'indent'; ... }        // </> 缩进
```

## Voice 语音输入

### 什么是 Voice？

语音输入功能，让用户通过麦克风与 Claude 对话。

- **门控**：`feature('VOICE_MODE')` + GrowthBook `tengu_amber_quartz_disabled`
- **入口**：`voice/voiceModeEnabled.ts`
- **认证要求**：必须使用 Anthropic OAuth（不支持 API Key、Bedrock、Vertex）

### 可用性检查

```typescript
// voice/voiceModeEnabled.ts

// GrowthBook 开关检查
function isVoiceGrowthBookEnabled(): boolean {
  // 1. feature('VOICE_MODE') 编译时开关
  // 2. GrowthBook tengu_amber_quartz_disabled 未禁用
}

// 认证检查
function hasVoiceAuth(): boolean {
  // 必须有 Anthropic OAuth accessToken
  // 不支持：API Key、Bedrock、Vertex、Foundry
}

// 完整检查
function isVoiceModeEnabled(): boolean {
  return hasVoiceAuth() && isVoiceGrowthBookEnabled()
}
```

### 技术栈

| 组件 | 说明 |
|------|------|
| `services/voice.ts` | 语音服务核心 |
| `services/voiceStreamSTT.ts` | 语音转文字流 |
| `hooks/useVoice.ts` | 语音 React Hook |
| `hooks/useVoiceEnabled.ts` | 语音可用性 Hook |
| `hooks/useVoiceIntegration.tsx` | 语音集成 Hook |
| `context/voice.tsx` | 语音 Context Provider |
| `vendor/audio-capture-src/` | 音频捕获原生模块 |

### 工作流程

```
1. /voice 命令启动
     │
2. 检查认证和门控
     │  hasVoiceAuth() + isVoiceGrowthBookEnabled()
     │
3. 初始化音频捕获
     │  vendor/audio-capture 原生模块
     │
4. 实时语音转文字
     │  voiceStreamSTT → claude.ai voice_stream endpoint
     │
5. 转录文本作为用户输入
     │
6. Claude 回复
```

## 子系统对比

| 特性 | Buddy | Vim | Voice |
|------|-------|-----|-------|
| 门控 | 编译时 | 无（内置） | 编译时 + 运行时 |
| 功能性 | 装饰/情感 | 编辑效率 | 输入方式 |
| 认证要求 | 无 | 无 | OAuth 必需 |
| 状态管理 | 全局配置 | 纯函数状态机 | React Context |
| 原生依赖 | 无 | 无 | 音频捕获模块 |
| 复杂度 | 中（6 文件） | 中（5 文件） | 高（7+ 文件 + 原生） |
