# 09 - UI 组件体系

## 概述

Claude Code 使用 **Ink**（React 终端 UI 框架）构建交互界面。项目对 Ink 进行了深度定制，实现了双缓冲渲染、虚拟滚动、鼠标选择等高级终端 UI 特性。

- **组件目录**：`components/`（150+ 组件）
- **Ink 定制**：`ink/`（深度修改的 Ink 框架）
- **屏幕组件**：`screens/`
- **入口**：`components/App.tsx`

## 组件层次结构

```
App.tsx
```

## 自定义 Ink 框架

### 架构概览

```
┌─────────────────────────────────────────┐
│                Ink 类                    │
│  管理 React reconciler + 渲染管线        │
│                                         │
│  ┌──────────┐    ┌──────────────────┐   │
│  │ reconciler│    │   DOM 树         │   │
│  │ (React    │───►│ (自定义 DOM     │   │
│  │ Fiber)    │    │  DOMElement)     │   │
│  └──────────┘    └────────┬─────────┘   │
│                           │              │
│                  ┌────────▼─────────┐   │
│                  │    Renderer       │   │
│                  │ DOM → 输出缓冲    │   │
│                  └────────┬─────────┘   │
│                           │              │
│              ┌────────────▼──────────┐  │
│              │   Screen Buffer       │  │
│              │ Cell + StylePool +     │  │
│              │ CharPool + Hyperlink   │  │
│              └────────────┬──────────┘  │
│                           │              │
│     ┌─────────────────────▼──────────┐  │
│     │        双缓冲 Frame             │  │
│     │  frontFrame ←→ backFrame       │  │
│     └─────────────────────┬──────────┘  │
│                           │              │
│              ┌────────────▼──────────┐  │
│              │   Diff 优化写入        │  │
│              │ writeDiffToTerminal   │  │
│              └───────────────────────┘  │
└─────────────────────────────────────────┘
```

### Ink 类

```typescript
class Ink {
  private readonly terminal: Terminal        // 终端抽象
  private scheduleRender: (() => void)      // 渲染调度器
  private readonly container: FiberRoot      // React Fiber 根
  private rootNode: dom.DOMElement          // DOM 根节点
  readonly focusManager: FocusManager       // 焦点管理

  private renderer: Renderer               // 渲染器
  private readonly stylePool: StylePool     // 样式池

  // 双缓冲帧

  private backFrame: Frame

  // 选择状态
```

## 渲染管线

### 1. React 更新

```
组件状态变更
     │
     ▼
React Fiber Reconciler
     │
     ▼
生成新的 DOM 树
     │
     ▼
调度渲染 (scheduleRender)
```

### 2. DOM → 输出缓冲

```
DOM 树
     │
     ▼
render-node-to-output.ts
  • 计算每个节点的布局（flexbox）
  • 应用样式（颜色、粗体等）
  • 生成 Cell 网格
     │
     ▼
Screen Buffer (screen.ts)
  • StylePool — 样式去重池
  • CharPool — 字符去重池
  • HyperlinkPool — 超链接池
```

### 3. 双缓冲 + Diff 写入

```
Screen Buffer
     │
     ▼
Back Frame（后台帧）
     │
     ▼
与 Front Frame（前台帧）比较
     │
     ▼
仅写入变化的部分 (writeDiffToTerminal)
     │
     ▼
交换 Front/Back Frame
```

### 优化策略

1. **双缓冲** — 避免闪烁
2. **Diff 写入** — 只更新变化的终端区域
3. **样式池** — 相同样式共享内存
4. **字符池** — 相同字符串去重
5. **虚拟滚动** — 只渲染可见消息
6. **异步调度** — 批量处理多次状态更新

## 鼠标选择

```typescript
// selection.ts
type SelectionState = {
  start: Position      // 选择起始位置
  end: Position        // 选择结束位置
  active: boolean      // 是否正在选择
  mode: 'char' | 'word' | 'line'  // 选择模式
}
```

- **单击拖拽** — 字符级选择
- **双击** — 词级选择
- **三击** — 行级选择
- 选中文本自动复制到剪贴板
