# 12 - 工具函数库

## 概述

`utils/` 目录是 Claude Code 最庞大的模块，包含 **564 个文件**，覆盖从 Bash 解析、Git 操作到加密、代理等几乎所有基础设施需求。

## 目录结构

```
utils/
├── bash/              (16) Bash 解析器、AST、Heredoc、补全
├── git/               (3)  Git 文件系统、配置解析、gitignore
├── shell/             (10) Shell 提供者、输出限制
├── hooks/             (17) Hook 执行引擎
├── settings/          (17) 设置加载、验证、缓存
├── permissions/       (24) 权限模式、规则、分类器
├── plugins/           (44) 插件市场、安装、策略
├── memory/            (2)  记忆类型与版本
├── skills/            (1)  技能变更检测
├── mcp/               MCP 工具函数
├── model/             模型选择、成本、Provider
├── messages/          消息处理工具
├── sandbox/           沙箱适配器
├── telemetry/         遥测工具
├── secureStorage/     密钥链、安全存储
├── computerUse/       Computer Use 工具
├── filePersistence/   文件持久化层
├── swarm/             Swarm/多 Agent 工具
├── ultraplan/         Ultraplan 功能
└── (200+ 独立文件)    各类独立工具函数
```

### 纯 TypeScript 实现

```typescript
// bash/bashParser.ts
// 生成兼容 tree-sitter-bash 的 AST

type TsNode = {
  type: string       // 节点类型
  text: string       // 源文本
  startIndex: number
  endIndex: number
  children: TsNode[]
}

function parseSource(
  source: string,
  timeoutMs?: number   // 默认 50ms
): TsNode | null

// 安全限制：
// - 50ms 挂钟超时
```
